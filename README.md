Engineering & Hardware Projects
Motorized Wheelchair Conversion

Repository: https://github.com/aravmehta20/wheelchair-conversion

Converted manual nursing home wheelchairs into fully electric wheelchairs using custom 3D-printed couplers (PETG-CF/TPU), motor drivers, lithium-ion battery packs, and joystick control.

GermGuard UV/Blue-Light Wristband

Repository: https://github.com/aravmehta20/germguard-wristband

A wearable device that detects handwashing events and emits controlled blue-light disinfection bursts; custom TPU enclosure designed for waterproofing and airflow.

Electrooculography (EOG) Eye-Control System

Repository: https://github.com/aravmehta20/eog-wheelchair-control

Eye-movement interface built with the BioAmp EXG Pill, TL074 op-amps, and ADS1015 analog-to-digital converters for wheelchair and HCI control.

Fall-Prevention Watch (Rice PATHS-UP Program)

Repository: https://github.com/aravmehta20/fall-prevention-watch

Wearable fall-detection prototype using IMU data and threshold-based motion classification, developed under Rice University doctoral mentorship.

UVC/Blue-Light Skin-Safe Disinfection Research

Repository: https://github.com/aravmehta20/uvc-skin-safety

Exploration of safe disinfection methods combining UVC, blue light, and timed bursts for antimicrobial effectiveness while maintaining dermatologic safety.

Hand-Gesture Recognition Model (CoreML)
